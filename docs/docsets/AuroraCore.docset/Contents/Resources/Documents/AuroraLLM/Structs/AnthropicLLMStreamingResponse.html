AnthropicLLMStreamingResponse

public struct AnthropicLLMStreamingResponse : Codable

Represents a streaming response from Anthropic’s LLM service.

This struct captures various components of the streaming response, including content deltas, content blocks, and usage statistics. It is designed to handle partial responses received incrementally during the streaming process.
